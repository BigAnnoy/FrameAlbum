❓ 使用帮助


📖 基本功能

  • 点击左侧目录树，浏览各年/月的照片
  • 点击照片卡片打开预览，支持键盘 ← → 翻页
  • 点击"➕ 导入"可从其他文件夹导入照片，自动去重

☑ 多选 & 删除

  • 点击工具栏"☑ 选择"进入多选模式
  • 长按照片卡片也可触发多选模式
  • 勾选后点击"🗑 删除"批量删除

🖱 右键菜单

  • 右键点击照片卡片：预览 / 打开文件 / 删除

⌨ 快捷键

  • F5 — 刷新相册
  • ESC — 关闭当前弹窗
  • ← / → — 预览上/下一张

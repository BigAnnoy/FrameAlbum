# 照片整理工具

> 自动扫描并按拍摄日期整理电脑中的照片

## 功能特点

- ✅ 自动读取 EXIF 拍摄日期，精准分类
- ✅ 按 `年份/年月` 双层目录结构整理
- ✅ 支持 JPG、PNG、HEIC、WebP、BMP、GIF、TIFF 等图片格式
- ✅ 支持 MP4、MOV、AVI、MKV 等视频格式
- ✅ 视频缩略图（需安装 ffmpeg，未安装则显示图标）
- ✅ MD5 去重（源内重复 + 目标重复检测）
- ✅ 无损复制，原始照片完整保留
- ✅ **现代化 UI**：Win11 风格主题、卡片悬停效果、选中高亮

## 环境要求

- Python 3.8+

## 安装依赖

```bash
pip install -r requirements.txt
```

### 可选：现代化主题

程序使用 `ttkbootstrap` 提供现代化主题效果。如果未安装，程序会自动降级使用默认 ttk 样式，不影响功能使用。

```bash
pip install ttkbootstrap
```

**主题效果：**
- 扁平化设计，Win11 风格
- 卡片圆角 + 阴影
- 按钮悬停效果
- 选中高亮反馈


### 可选：视频缩略图支持

如果希望视频显示真实缩略图（而非图标），需要安装 [ffmpeg](https://ffmpeg.org/)。

**快速安装（Windows）：**

1. 双击运行 `install_ffmpeg.bat`，按提示选择安装方式：
   - **方案 A**（推荐）：使用 Winget 一键安装
   - **方案 B**：手动下载并配置

2. 重启照片整理工具

**验证安装：**
```bash
ffmpeg -version
```

**手动安装步骤（如上述脚本不可用）：**

- Windows: 下载 https://www.gyan.dev/ffmpeg/builds/，解压并添加到系统 PATH
- Mac: `brew install ffmpeg`
- Linux: `sudo apt-get install ffmpeg`

> 💡 不安装 ffmpeg 也可正常使用，视频只会显示 🎬 图标。

## 运行

```bash
python main.py
```

或双击 `启动.bat`

## 使用方法

1. 点击"源文件夹"旁的按钮，选择包含照片/视频的文件夹
2. 点击"目标文件夹"旁的按钮，选择整理后照片的存放位置
3. 点击「▶ 预览」查看将要整理的内容
4. 点击「✓ 开始整理」按钮，等待完成

## 日志

程序运行时会在 `logs/` 目录生成日志文件：

```
logs/
└── photo_organizer_20260323.log  # 按日期命名：photo_organizer_YYYYMMDD.log
```

日志包含：
- 程序启动/关闭记录
- 扫描进度和统计
- 整理结果（成功/失败/重命名数量）
- 错误和警告信息

## 目录结构示例

整理后的照片会按以下结构存放：

```
目标文件夹/
└── 2024/
    └── 2024-03/
        ├── IMG_0001.jpg
        ├── IMG_0002.jpg
        └── video_001.mp4
```

说明：
- 第一级目录：年份（YYYY）
- 第二级目录：年月（YYYY-MM）

## 注意事项

- 整理过程可选择 **复制** 或 **移动**，默认为复制
- 无 EXIF 信息的照片会使用文件的修改时间
- 损坏或无法识别的文件会被跳过并记录
- 整理前预览界面可删除重复或不需要的文件

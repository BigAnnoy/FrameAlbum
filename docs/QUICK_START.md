### 🔧 技术细节

- **MD5 重复检测**：基于文件内容计算 MD5 哈希值
- **日期识别**：优先读取 EXIF 信息，备用文件修改时间
- **异步扫描**：后台线程扫描，不阻塞 UI
- **缩略图缓存**：LRU 策略，最多缓存 500 张
- **配置保存**：保存到 `~/.photo_organizer/config.json`

### ❓ 常见问题

**Q：如果提示找不到 Python？**
A：确保 Python 已安装且加入 PATH，或者用完整路径启动

**Q：重复检测不工作？**
A：确保源目录中的重复文件内容完全相同

**Q：如何重置配置？**
A：删除 `~/.photo_organizer/config.json` 文件，重启程序会重新初始化

**Q：能否测试多次？**
A：可以！每次导入都可以选择新的源文件夹或重复操作

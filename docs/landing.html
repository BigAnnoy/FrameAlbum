技术架构

轻量、本地、
开放的技术栈

没有 Electron 的臃肿，没有云依赖的不安。纯 Python + Web 技术，可随时审查、随时修改。

01
PyWebView 窗口层
原生系统窗口包裹 Web UI，调用系统文件对话框，无 Electron 依赖，内存占用极低。
02
Flask REST API 后端
33KB 核心 API 服务，20+ 端点，含缩略图缓存、视频元数据、文件操作全覆盖。
03
原生 HTML/CSS/JS 前端
零框架依赖，模块化 JS 设计（API / 相册浏览 / 导入对话框 / 照片多选各司其职）。
04
SQLite + SQLAlchemy 数据层
本地 SQLite 数据库存储配置与元数据；JSON 记录文件追踪已导入 MD5，双重保险。
05
Pillow + pillow-heif 图像处理
缩略图生成、HEIC/TIFF 格式转换、调色板透明图处理，全部本地计算，不上传任何图片。
06
FFmpeg 视频
视频缩略图提取、时长/分辨率/编码元数据读取，HTTP Range 协议支持任意 Seek。

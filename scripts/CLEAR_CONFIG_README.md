# 📸 相册管理 - 清除配置脚本

## 用途

此脚本用于清除相册管理应用的所有配置信息，包括：
- 相册路径设置
- 导入历史记录
- 应用程序设置（如默认导入模式）

执行此脚本后，应用程序将恢复到首次启动状态，下次启动时需要重新选择相册路径。

## 位置

脚本位置：`E:\PhotoManager\scripts\clear_config.py`

## 使用方法

### 1. 直接运行脚本

```bash
python clear_config.py
```

### 2. 自动化运行（跳过确认）

```bash
echo y | python clear_config.py
```

## 执行流程

1. 脚本启动后会显示操作说明和警告信息
2. 提示用户确认是否继续操作
3. 如果用户确认，将执行以下操作：
   - 检查配置文件是否存在
   - 删除配置文件 `config.json`
   - 如果配置目录为空，删除配置目录
4. 显示操作结果

## 配置文件位置

配置文件默认存储在：`E:\PhotoManager\.config\config.json`

## 注意事项

- ⚠️ 此操作不可逆，请谨慎使用
- 执行后需要重新设置相册路径
- 如果应用程序正在运行，建议先关闭再执行此脚本

## 日志信息

- 操作步骤
- 成功或失败的状态
- 错误信息（如果发生）
